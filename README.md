## Introduction
Welcome! This repository contains a game with a simple swipe mechanic.

## Dependencies

- [Zenject](https://github.com/modesttree/Zenject)
- [DoTween](http://dotween.demigiant.com/)
- [UniTask](https://github.com/Cysharp/UniTask)
- [Naughty Attributes](https://github.com/dbrizov/NaughtyAttributes)
- [Toolbar Extender](https://github.com/marijnz/unity-toolbar-extender)
- Addressable

## Level Generation
If you want to create levels in this system, you need to follow the path: 
- Gem Swipe (in the Menu Bar) -> Level Generator -> Level Generate

Save the level you created to the Levels folder and don't forget to select 'Addressable'!

Play this game with unlockable boards!
